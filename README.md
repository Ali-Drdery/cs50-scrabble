# CS50 Scrabble

This project was completed as part of Harvard CS50x.

## Description
A C program that simulates the scoring system of the Scrabble game.

The program takes words from two players, calculates the score of each word based on letter values, and determines the winner.

## Technologies Used
- C
- CS50 Library

## Concepts Learned
- Arrays
- Functions
- String manipulation
- ASCII values
- Conditionals and loops

## Features
- Supports uppercase and lowercase letters
- Ignores non-letter characters
- Calculates scores automatically
